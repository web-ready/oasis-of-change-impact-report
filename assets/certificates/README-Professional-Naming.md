## Naming Standards

### Folder Names:
- Use numbered prefixes (01-, 02-, etc.) for consistent ordering
- Use hyphens instead of spaces
- Remove abbreviations from folder names
- Keep names descriptive but concise

### File Names:
- Start with date (YYYY-MM-DD)
- Include organization name
- Include project type/name
- Include location
- Include impact metrics (trees planted, CO2 offset)
- Use hyphens for word separation
- Use consistent capitalization
- Fix typos (e.g., "tazania" → "Tanzania")

### Benefits:
- ✅ Professional appearance
- ✅ Easy sorting by date
- ✅ Clear organization identification
- ✅ Consistent formatting
- ✅ Searchable and filterable
- ✅ Self-documenting file names
