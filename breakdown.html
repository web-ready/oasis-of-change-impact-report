About This Data

Tree-Nation counts are fetched live from the API when available. If the API is unreachable (e.g. due to CORS restrictions), data from the last manual update (—) is displayed instead. For partners with shared plantings, the shared allocation with the Web-Ready by Oasis of Change, Inc. forest is excluded from additive totals; any additional trees in that partner forest are counted as additive. Legacy partner counts are fixed and cannot be independently verified. 1

Methodology & notes

  1. 1 Tree-Nation does not set CORS headers on their public API. In browsers, this means the live data fetch may be blocked depending on your network and browser settings. When live data is unavailable, the page falls back to the most recent manually verified counts.
