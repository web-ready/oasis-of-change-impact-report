# Oasis of Change Impact Report

**Transparent report of tree-planting impact and legacy programmes**  
Last updated July 2, 2025 • Vancouver, Canada

## What’s Inside

This repository contains a single, self-contained HTML report that presents:

- **Verified plantings** via Tree-Nation certificates (five reforestation projects + gift batches)  
- **Legacy partner totals** from closed Tero & Refoorest programmes  
- An **executive summary** with visual bars showing certified vs. legacy counts  
- **Tabbed tables** for easy browsing of both categories  
- A clear footnote on data provenance and our plan to route all future plantings through Tree-Nation

## File

- **`tree-impact-report.html`**  
  The main HTML file, styled with Tailwind CSS (via CDN) and lightweight JavaScript for tab toggling.

## How to Use

1. **Clone this repo**  
   ```bash
   git clone https://github.com/your-org/oasis-of-change-impact-report.git
   cd oasis-of-change-impact-report
````

2. **Open the report**
   Simply open `tree-impact-report.html` in any modern web browser—no build or server required.

3. **Update as needed**

   * Change the “Last updated” date and location in the header.
   * Edit tree counts, certificate numbers, or bar widths directly in the HTML.
   * Add new sections if additional environmental programs are included.

## Data Sources

* **Tree-Nation certificates** (e.g. `TN-MST-2025-Q2-001`, `TN-SPES-2025-06-15`, etc.)
* **Legacy dashboards** from Tero and Refoorest browser extensions (2019–2024)
