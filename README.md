# Oasis of Change Impact Report

**Transparent report of tree-planting impact and legacy programmes**  
Last updated July 2, 2025 • Vancouver, Canada

## What’s Inside

This repository contains a single, self-contained HTML report that presents:

- **Verified plantings** via Tree-Nation certificates (five reforestation projects + gift batches)  
- **Legacy partner totals** from closed Tero & Refoorest programmes  
- An **executive summary** with visual bars showing certified vs. legacy counts  
- **Tabbed tables** for easy browsing of both categories  
- A clear footnote on data provenance and our plan to route all future plantings through Tree-Nation
